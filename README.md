minscript
=========

MinScript is a small and compact language that compiles to JS. It has a lisp
syntax.


# Language Reference

## Atoms

An atom is something that represents a value.

- Int
```clojure
(let foo 12) ;=> 12
```
- Float
```clojure
(let foo 1.12) ;=> 12
```
- Strings
  `(let foo 'foo') ;=> 'foo'`
